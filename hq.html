HQ runs locally on your machine. To use it, you need OpenClaw installed and the Agent Arena server running.

STEP 01

Install OpenClaw

OpenClaw is the runtime that powers your AI agents. Get it at github.com/openclaw/openclaw or:

npm install -g openclaw
STEP 02

Install an Agent

Pick an agent and run the installer. It sets up the agent's workspace, identity, and connects it to your gateway.

curl -fsSL raw.githubusercontent.com/h53674242-create/agent-arena/main/installer/install.sh | sh -s founder-agent
STEP 03

Start the Arena Server

Clone the repo and start the local HQ server. Then refresh this page.

cd agent-arena && node server.js
